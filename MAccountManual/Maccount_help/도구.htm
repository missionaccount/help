도구 - DB 최적화 / 파일정리

현 프로그램은 MDB 파일로 구성되어 있습니다. (Access 2002)

MDB 파일은 각종 log 정보가 기록되므로 한 달정도 쓰게 되면 용량이 매우 커지게 됩니다.

따라서 관리자는 이 메뉴를 사용하여 최적화를 시켜주어야 합니다.

(추천 : 2주 또는 1달에 한 번)

　

데이타최적화 버튼을 누르기 전에는 다른 사용자가 회계시스템을 사용하고 있으면 안됩니다.

관리자만 접근한 상태에서 해주어야 합니다.

　

데이타베이스 최적화 단추를 누르면 최적화 됩니다.

　

　

각 사용자의 컴퓨터에는 각종 리포트 파일이 존재합니다.
엑셀파일과 HTM 파일, Txt 파일

(프로그램폴더\ReportData)

상기 폴더내에 있는 것은 보안이 필요하므로 꼭 필요한 것은 복사해 놓으시고 아래 버튼을 눌러 비워주시기 바랍니다.

　

MDB 파일을 수정하여 수동으로 자료를 업데이트 시키려면...

회계시스템 초기에 MDB 파일을 직접 다루어서 자료를 넣어두려면 아래와 같이 하시면 됩니다.

관리자가 MDB 수정모드(비밀번호 0000) 을 선택하면 (다른 사용자 사용금지, 관리자만 접속된 상태)

서버폴더에 있는 Account.mdb  파일의 비밀번호가 0000으로 바뀝니다.

그런다음 이 파일을 작업하기 좋은 폴더에 복사해 둡니다.

곧바로

MDB 프로그램 암호부여 (MDB 수정후 필수) 버튼을 누릅니다.

이 버튼을 눌러야 회계 시스템에 맞는 MDB 파일로 변합니다.

(주의 : 비밀번호 0000 인 상태는 현재 프로그램에 맞지 않습니다. 반드시 암호를 부여해 주어야 합니다)

　

작업하기 좋은 폴더에 복사한 MDB 파일의 암호는 0000 이므로 이 파일을 수정한 후 다시

서버폴더에 파일을 복사하여 겹쳐쓴 후 (겹쳐쓸 때 원본은 반드시 사본으로 복사해 놓을 것)

MDB 프로그램 암호부여 버튼을 누르면

현 시스템에서 사용할 수 있는 상태가 됩니다.
