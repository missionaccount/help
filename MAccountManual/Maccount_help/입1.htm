전표의 입력

전표를 입력하려면 전표편집 - 전표입력을 선택하면 됩니다.

　

　

전표구분에서 7가지 사항 중 해당사항을 선택하면 적합한 전표입력창이 나타납니다.

　

전표구분 설명

1. 자산(현금/예금) 증가 : 주로 후원금 등의 수입만 있는 즉, 입금전표만 사용할 때 편리하게 사용하실 수 있습니다.

2. 자산(현금/예금) 감소 : 주로 지출만 있는 즉, 출금전표만 사용할 때 편리하게 사용하실 수 있습니다.

3. 현금->예금입금 : 이것은 거래는 아니나 현재 가지고 있는 현금을 어느 은행에 입금했는 지 기록하는 곳입니다.

4. 예금->현금인출 : 이것은 거래는 아니나 현재 가지고 있는 예금에서 현금으로 출금한 내역을 기록하는 곳입니다.

5. 예금계좌간 이동 : 이것은 거래는 아니나 은행계좌 사이에서 이동시킬 때 사용하는 곳입니다.

6. 내부거래(계정간) : 이것은 거래는 아니나 현재 계정사이의 금액잔액을 변동시킴으로 계정간 금액을 조정할 수 있습니다.

7. 대체전표 : 입금, 출금 전표 모두를 사용할 수 있으며 대체거래를 기입할 수 있는 란입니다.

대체거래가 발생하는 전표는 모두 이곳에서 작성하면 됩니다. 또는 기말정리사항에서도 이곳에서 작성하면 됩니다.
