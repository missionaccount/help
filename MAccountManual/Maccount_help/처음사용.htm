프로그램을 처음 사용하실 때에는 반드시 환경설정을 해야 합니다.

1. 환경설정

환경설정 메뉴 버튼을 누르면 환경을 설정하실 수 있습니다.

　

2. 관리자 설정
　

프로그램의 관리자의 정보를 입력합니다. (관리자는 0 등급입니다)

관리자명을 입력하실 때 가능하면 영문으로 입력하여 주십시오.
(ID와 패스워드를 입력합니다, 패스워드는 두 번 입력하게 되어있습니다)

　

3. 관리자외 사용자 설정

각 사용자의 아이디와 패스워드를 입력합니다. (이것은 관리자만 고칠 수 있습니다)

권한을 1은 대개 단체장에게 부여합니다. (사용자명에 단체장 이름을 입력하면 됩니다-> 가능하면

영문으로 입력하십시오)

읽기전용이라는 것은 각 내용들은 모두 볼 수 있지만 입력할 수는 없으며, 특히 관련 리포트는 모두

볼 수 있습니다.

　

입력자 권한  2등급은 입력은 가능하지만 여러가지 설정은 할 수 없습니다.

　

3등급인 읽기전용은 주로 다른 선교사의 후원금이나 통계는 볼 수 없으며 간단한 후원자 리스트 같은 것은

보실 수 있습니다.

　

4. 프로그램 제목을 입력합니다. (해당 단체 이름을 적어주면 됩니다)

작업대상 최대선교사수는 전표처리를 원활하게 하기 위해서 현재 500명의 선교사가 있다면

600 정도로 해두시면 됩니다. (관리자가 수시로 고쳐서 사용하실 수 있습니다)

　

5. 일괄처리에 관한 권한을 부여하는 것입니다.
　

주로 관리자가 일괄처리를 하지만 데이타 입력자인

2등급에도 그 권한을 부여하고 싶으면 여기서 선택하시면 됩니다. (한시적으로 부여해도 됨)

일괄처리에는 행정비 공제 일괄처리, 경조비, 보험처리등 이 있습니다.
 

　

6. 각종 테이블 상의 행의 색깔을 지정하는 것입니다.
(주로 읽기모드에서 적용됩니다)

적용한 후 ok 버튼을 꼭 눌러주어야 적용됩니다.

주로 미색을 적용하십시오. 너무 진한 색을 적용하면 글자가 보이지 않습니다.

행의 색상 예

　

　

모든 것을 지정하셨으면 저장버튼을 누르고 닫기 버튼을 눌러서 빠져나옵니다.

(반드시 저장버튼을 누르셔야 합니다.)
