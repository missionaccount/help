설치방법

본 프로그램은 네트워크 상에서 회계관리를 할 수 있도록 한 프로그램입니다. 하지만 경우에 따라서는 네트워크가 연결되지 않은 컴퓨터에서 1대에서만 활용할 수도 있습니다.

* 현 프로그램은 XP 상에서 원활하게 작동되도록 되어 있습니다. (적어도 window2000 이상에서 사용 가능합니다) 

A. 네트워크상에서 설치방법

1. 프로그램 CD를 사용자(클라이언트) 컴퓨터에 넣고 MaccountSetup.exe 를 더블클릭하여 설치를 시작합니다.

설치가 끝나면 아래 프로그램이 설치됩니다. 그러나 현재 상태로 사용하실 수 없습니다. 서버에 DB 데이타가
설치되어야 사용가능합니다.
 

　

2. 각 클라이언트 컴퓨터에서 MaccountSetup.exe 를 설치하고 나면 이제 서버에 데이타베이스를 복사해 넣어야 합니다.

CD에 있는 아래 폴더(서버에둘폴더)를 서버에 해당하는 컴퓨터에 복사한 후 명칭을 알기쉽게 바꾸십시오.

(예 : 2005MAccount)

  

명칭을 2005MAccount로 바꾼 후 마우스오른쪽 버튼을 해당 폴더에서 누르고 속성을 선택하면  공유할 수 있도록 등록정보가 나옵니다.

폴더공유와 내파일을 변경할 수 있음을 선택한 후 확인을 눌러줍니다.

 

### Notice ### 

* 서버에 해당하는 컴퓨터에는 Maccountsetup 프로그램을 설치할 필요가 없습니다. 단지 서버에 둘 폴더만 복사하여 놓고 이름을 적당히 바꾸어 놓은 다음 공유시키면 됩니다.

* 서버에 해당하는 컴퓨터가 win98 일 경우에는 네트워크 기능이 원활하게 동작하지 않을 수 있습니다. 가능하면 같은 윈도우시스템으로 해주시고 window xp 를 추천합니다.

　

이제 Maccount 단축아이콘을 눌러서 프로그램을 실행해 봅니다.

　

아래는 현재 프로그램과 서버의 DB 를 연결시키는 작업입니다.

* 서버의 2005MAccount 폴더를 찾으면 그 폴더를 선택해 줍니다. (드라이버는 Z로 잡아준다)

(주의사항 : 현재 시스템은 네트워크 폴더를 Z: 로 잡아주게 되어 있습니다.
만일 Z:를 다른 네트워크 폴더가 사용하고 있으면 로그인창에서 끊기 버튼을 눌러서 z: 디렉토리를
끊고 회계시스템의 폴더를 Z: 로 지정해 주어야 합니다.)

* 동그라미 부분에 있는 내용을 수정적용할 텍스트박스에 넣고 수정적용을 눌러주시면 됩니다. (동그라미 부분을
마우스로 한 번 클릭하면 모두 선택됩니다. 이 상태에서 Ctrl + C 를 하시면 복사되며 이것을 수정적용할 텍스트박스에
넣고 Ctrl + v 하시면 갖다 붙이실 수 있습니다.

취소버튼을 눌러서 빠져나옵니다.

아래 그림처럼 연결될 네트워크 또는 경로에 복사한 글자를 입력해주고 수정적용 단추를 누르시면 됩니다.

　

그런 다음 로그인을 해주시면 됩니다.
　

맨 처음 입력하실 때에는
아이디와 패스워드 모두 점(.) 을 찍어주면 됩니다.

　

　

B. 네트워크를 사용하지 않고 단독으로 사용할 경우

회계시스템을 단독으로 사용하게 될 경우에는 폴더를 공유하지 않아도 됩니다. 자신의 하드디스크에 저장하시면 됩니다.

(루트디렉토리에 폴더를 저장하기를 추천합니다.  예 : C:\2005MAccount  )
　

단독으로 사용하실 대에는 DB가 연결될 경로를 적어주시고
네트워크 사용안함에 체크를 한다음
수정적용 단추를 눌러 주시면 됩니다.

그런다음 아이디와 패스워드는 초기에는 점(.)만 찍으면 들어갈 수 있도록 했습니다.
