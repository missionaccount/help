선교사별 후원통계 보기

　

각 선교사의 후원자 통계입니다.

후원자 상세보기 : 후원자 리스트를 보여줍니다.

월별금액 상세보기 :  선교사별 월별 후원금액 통계를 보여줍니다.

　

위의 내용을 엑셀로 저장시 현재 상태에서 확인 버튼을 누르면 됩니다.

(HTM 저장시에는 후원자통계저장 HTM을 선택하시고 확인을 누르시면 위의 내용이 저장됩니다)

　

만일 일괄처리로

선택된 선교사의 후원자리스트를 모두 저장하려면

현 상태에서 확인 버튼을 눌러주시면 됩니다.

(메일로 발송하려면 아웃룩익스프레스에 메일 계정이 세팅되어 있어야 합니다.

또한 메일발송을 먼저 선택후 좌측의 해당사항을 선택하시고 확인버튼을 누르시면 됩니다)

　

주의 : 선교사 이메일에 메일이 있는 것만 발송됩니다. (기초정보-계정정보)

메일에 대한 세부사항은 도움말에서 메일보내기를 참조하십시오.
